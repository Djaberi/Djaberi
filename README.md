### 🔭 Featured projects

**[SIEM Detection Lab](https://github.com/Djaberi/siem-detection-lab)**
A hands-on Splunk lab covering the full detection lifecycle — phishing, VPN auth
anomalies, DLP exfiltration, and malware behavior. Each detection ships with
sample data, an SPL search, a Sigma rule, and a matching IR playbook, all mapped
to MITRE ATT&CK.

**[Sigma Detection Pack](https://github.com/Djaberi/sigma-detection-pack)**
A curated set of vendor-neutral Sigma detection rules across 9 ATT&CK tactics —
credential dumping, encoded PowerShell, persistence, lateral movement, and more.
Structurally validated in CI with an auto-generated coverage matrix.

**[Home SOC Lab](https://github.com/Djaberi/home-soc-lab)**
A segmented virtual SOC (FortiGate + Security Onion + Splunk) for practicing
detection engineering and incident response end to end. Includes architecture
docs and ticket-style investigation write-ups of simulated attacks.

**[SOC Automation Toolkit](https://github.com/Djaberi/soc-automation-toolkit)**
Python utilities for SOC triage — IOC extraction, phishing header analysis,
VirusTotal enrichment, and auth-log brute-force detection. Installable CLI with
a pytest suite and CI across Python 3.10-3.12.
